FreeIPA Server

Setup Hostname 

        
            sudo vi /etc/hosts
            ip.address.of.server hostname.server.com
            :wq

Installing FreeIPA LDAP Server packages

enable repo and sync it

        
            yum module enable idm:DL1
            yum distro-sync

install packages

        yum -y install ipa-server ipa-server-dns

Installing FreeIPA LDAP Server

        
            sudo ipa-server-install --setup-dns --hostname=$(cat /etc/hostname)

Using the firewall

        
            sudo firewall-cmd --add-service={freeipa-ldap,freeipa-ldaps,dns,ntp} --permanent; sudo firewall-cmd --reload

Getting a Kerberos ticket for admin

        
            kinit admin
