VirtualBox Command Line Interface(CLI)

you want to be in the C:\"Program Files"\Oracle\VirtualBox directory before you do any commands

List VMs 

        
            vboxmanage.exe list vms

get Mac address of VM

        
            vboxmanage.exe showvminfo --details [vm_uid] | findstr MAC

shutdown VM

        
            vboxmanage controlvm [vmname] poweroff soft

See VM power status

        
            vboxmanage showvminfo "vmname" | findstr "running (since"
