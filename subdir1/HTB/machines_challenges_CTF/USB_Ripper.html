Hackthebox USB-Ripper Challenge

Steps

  • unzip USB_Ripper.zip
  • cd usb-ripper
  • sudo -H python3 -m pip install --upgrade usbrip
  • sudo usbrip events violations auth.json --file syslog
  • select Terminal stdout
  • copy serial number, and decrypt the md5 hash
  • Submit the decrypted hash to hackthebox
