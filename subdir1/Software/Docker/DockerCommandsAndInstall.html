Packages

Install these packages to install Docker CE: docker-ce, docker-ce-cli, containerd.io

Add yourself to the docker group

sudo usermod -aG docker $(whoami)

See docker images

docker images

Remove docker images

docker rmi image_id

Start, Stop, Kill container

docker (start|stop|kill) containerID

Use a conatainer, edit it, and then update it

docker commit containerID username/imageName

Get number of containers running, paused, and/or stopped

docker info

login to docker repository

docker login

Create a docker container from an image

docker run -it -d imageName

Accessing a running container

docker exec -it containerID bash

Push image to dockerhub

docker push username/imageName

Build Dockerfile

docker build /path/to/docker_file

Monitor docker in production

docker stats

or

docker events

Install Docker Machine

Docker Machine is used to setup a docker swarm. To install docker-machine run this command: 

    base=https://github.com/docker/machine/releases/download/v0.16.0 ; curl -L $base/docker-machine-$(uname -s)-$(uname -m) >/tmp/docker-machine ; sudo mv /tmp/docker-machine /usr/local/bin/docker-machine ; chmod +x /usr/local/bin/docker-machine
