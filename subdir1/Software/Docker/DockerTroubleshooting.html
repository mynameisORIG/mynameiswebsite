Docker Troubleshooting

Config File

When installing Docker on Linux, it usually doesn't come configured with the config file which is located in /etc/docker/daemon.json. To set that run the following command: echo {} > /etc/docker/daemon.json; dockerd --config-file /etc/docker/daemon.json as root.

Do this if you receive this error: failed to start daemon: Error initializing network controller: Error creating default "bridge" network: Failed to program NAT chain: ZONE_CONFLICT: 'docker0' already bound to a zone 

    
        sudo firewall-cmd --permanent --zone=trusted --remove-interface=docker0
        sudo firewall-cmd --reload
        sudo systemctl restart docker
